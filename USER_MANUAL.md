## Amanah User Manual

This guide explains how to use Amanah as a parent or child.

## 1. What Amanah is

Amanah is a family savings planner with two user experiences:

- **Parent account**: manage children, savings goals, and contributions
- **Child account**: view goals, save toward personal targets, and interact with a virtual pet

All savings values are shown in **BHD**.

## 2. Getting started

### Sign up

1. Open the Amanah home page.
2. If the screen shows **Welcome Back**, click **Register**.
3. Enter:
   - full name
   - email address
   - password
   - account type: **Parent** or **Child**
4. Click **Create Account**.

### Sign in

1. Enter your email address and password.
2. Click **Sign In**.
3. After login:
   - parent users go to the main dashboard
   - child users are redirected to the child view automatically

## 3. Parent guide

### Parent dashboard

The dashboard shows:

- total family savings
- number of children
- overall progress toward savings goals
- a list of child cards

Use **Add Child** to create a new savings profile.

### Add a child

1. Open **Add Child**.
2. Fill in:
   - child name
   - date of birth
   - savings goal name
   - target amount
   - monthly contribution
   - target date
3. Click **Save Child**.

After saving, the child appears on the dashboard.

### Open a child profile

From the dashboard, select a child to view the detail page.

On the child page you can:

- view the goal overview
- review savings history and charts
- add a contribution
- edit child and goal details
- pause or resume automatic saving
- delete the child profile

### Add a manual contribution

1. Open the child profile.
2. Click **Add Contribution**.
3. Enter the amount.
4. Optionally enter a note.
5. Click **Add**.

The contribution is added to the child’s savings history.

### Edit child details

1. Open the child profile.
2. Click the **edit** icon.
3. Update the child name, date of birth, goal name, target amount, or target date.
4. Click **Save Changes**.

### Pause or resume a goal

1. Open the child profile.
2. Click **Pause** to stop automatic contributions.
3. Click **Resume** to start them again.

When a goal is paused, a banner appears on the page to show automatic saving is stopped.

### Delete a child

1. Open the child profile.
2. Click the **delete** icon.
3. Confirm deletion.

Deleting a child permanently removes that profile and its savings history.

## 4. Child guide

### First login and pet setup

The first time a child opens the child view, they are asked to:

1. choose a pet
2. give the pet a name

Available pets include bunny, cat, dragon, fox, and dog.

### Child home screen

The child home screen shows:

- total saved amount
- current coin balance
- number of personal goals
- current level
- pet preview and mood
- quick links to goals, pet room, and store

### Create a personal goal

1. Open **My Goals**.
2. Click **New Goal**.
3. Choose a goal type.
4. Enter the target amount.
5. Select a target date.
6. Click **Let’s Go!**

The goal will appear in the goal list with progress tracking.

### Save toward a personal goal

1. Open **My Goals**.
2. Select **Save!** on a goal.
3. Enter an amount or choose a quick amount.
4. Confirm the contribution.

Completed goals display a success state when the target is reached.

### Delete a personal goal

1. Open **My Goals**.
2. Use the delete icon on the goal card.
3. Confirm deletion.

This also removes that goal’s savings history.

### Use the pet room

Open **Pet** to interact with the child’s pet.

Available actions:

- **Feed**
- **Play**
- **Pet**

These actions increase happiness, but each one has a short cooldown before it can be used again.

The pet room also shows:

- happiness level
- current evolution stage
- equipped accessories
- evolution journey
- room add-ons unlocked through progress

### Pet stages and growth

The pet evolves as total savings increase. The stages progress from egg to more advanced forms as savings milestones are reached.

### Coins and the pet store

- Children earn **100 coins for every 1 BHD saved**.
- Open **Pet Store** to spend coins on items.
- Items are grouped into **Hats**, **Outfits**, and **Toys**.

To use the store:

1. Open the store.
2. Select a category.
3. Buy an item if you have enough coins.
4. Equip an owned item.
5. Remove an equipped item if desired.

## 5. Troubleshooting

### I cannot sign in

- Make sure the email and password are correct.
- If you do not have an account yet, use **Register** first.

### I signed in as a child and do not see the parent dashboard

This is expected. Child accounts are redirected to the child experience.

### My pet is missing

If it is the first visit, complete pet setup first.

### I cannot buy a store item

You may not have enough coins yet. Save more money to earn more coins.

### My pet actions are disabled

Feed, play, and pet actions use cooldowns. Wait briefly and try again.

### My pet items or progress are not showing on another device

The child pet system is stored in the browser on the current device. For best results, keep using the same browser/device.

## 6. Quick summary

- Parents manage children, goals, and contributions.
- Children manage personal goals and interact with a virtual pet.
- Savings progress helps children earn coins and grow their pet.
